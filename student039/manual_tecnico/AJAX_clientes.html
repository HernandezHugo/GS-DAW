Clientes

Los documentos relacionados con AJAX para la busqueda de clientes son:

Script AJAX

Buscamos el cliente por nombre. Obtenemos un resultado por cada carácter que introducimos en el input. Si existe un cliente con el nombre que escribimos en el input, se escribiran sus características en la página.

getClients.php

Obtenemos el valor del input que nos enviar el script AJAX. Después, comprobamos que existe algún cliente que contenga el valor recibido y lo enviamos al script para que enseñe sus características.
