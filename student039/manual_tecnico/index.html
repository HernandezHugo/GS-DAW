Bienvenido

En esta página encontrarás las funcionalidades del proyecto "Sito Hotel".

Antes de ver cualquier funcionalidad tienes que saber como se han gestionado los archivos.

Un rápido vistazo a las carpetas.

Cada carpeta tiene asignado el nombre según la funcionalidad o finalidad que hayan sido diseñados.

AJAX: Contiene todos los archivos que emplean AJAX; pedir datos a terceros, obtención de datos sin refresco de página.

Cookies: Contiene todos los archivos que emplean cookies.

CSS: Contiene el archivo CSS, donde se motifica elementos puntuales.

DB: Contiene todos los archivos que interactuan con la base de datos.

Forms: Contiene todos los formularios.

img: Contiene todas las imagenes necesarias para mostrar en tarjetas y portada.

img-weather: Contiene los iconos necesarios para enseñar el tiempo en la página de inicio.

sql: Contiene los procedemientos utilizados en la base de datos.

Templates: Contiene las plantillas de "header.php" y "footer.php", más las variantes que se añaden al "header.php": "nav-guest.php", "nav-admin.php" y "nav-client.php".

user_pfp: Contiene todas las imagenes de perfil de los "users" (admins).

Páginas principales: páginas que se construyen por templates y forms o contenido propio.
