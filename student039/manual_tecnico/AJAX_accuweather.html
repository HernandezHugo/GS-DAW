AccuWeather

Los documentos relacionados con AJAX y la petición a APIs de terceros son:

Scripts AJAX

Aplicamos AJAX para realizar una peticion a la API de AccuWeather. Esto hará que enseñe el tiempo de la zona del hotel y está representado en la esquina superior izquierda de la portada, en la página inicial

Con el script primero hacemos una peticion al archivo "check_accu_weather.php" y, dependiendo de la respuesta, haremos una peticion a la API de AccuWeather o utilizamos la información que hay en nuestra base de datos.

Una vez tengamos la información necesaria, procedemos a; guardar el JSON del tiempo en la base de datos, si hemos hecho la peticion a AccuWeather, y más tarde enseñarlo en la portada; o enseñarlo directamente en la portada, porque significará que no hemos hecho una peticion a la API.

PHP Docs

En el primer archivo php, comprobamos si existe un documento JSON con la información del tiempo en nuestra base de datos con un margen de tiempo de treinta minutos, respecto a la última peticion a AccuWeather.

Le enviamos una respuesta al script de la peticion para determinar si hacer una peticion a AccuWeather o no.

Obtenemos el JSON desde el script de AJAX. Después, obtenemos la fecha y hora actual (año-mes-dia hora-min), para añadirlo al nombre del documento, como información relevante.

Por último, hacemos un "INSERT" con la información relevante del documento: nombre del archivo, archivo(.json) y la fecha de obtencion.
