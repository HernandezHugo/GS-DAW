Clientes

Los documentos de SIUD (SELECT, INSERT UPDATE y DELETE)

Select

Obtienes los clientes de la base de datos.

Insert

Añades un cliente a la base de datos, tanto en admin como en "register".

Update

Te permite actualizar los clientes (como admin).

Delete

Eliminas el cliente (como admin).

Administradores

Los Administradores se insertarán en la base de datos media un formulario de registro de admin.

No hay una manera de ver a todos los administradores, no se podrán eliminar, ni se podrán actualizar, si no es en la propia base de datos.
