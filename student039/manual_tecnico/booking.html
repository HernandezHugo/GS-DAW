Realizar reserva

Display

Mostramos las categorias que estan disponibles según la fecha que introduzcas en los inputs que hay en el centro de la sección de "Booking".

PHP Docs

Este fragmento de código está incluido en el documento "Booking.php", porque inicializa la fecha inicial y final con las cookies (si no las rechazaste en la página inicial).

Cuando el cliente confirma la fecha inicial y la fecha final de su estadia en el hotel, se pasarán las fechas al procedimiento "039_caregories_to_show" para poder enseñar las categorias disponibles.

Este fragmento de código tambien está incluido en el documento "Booking.php".

Cuando el cliente clicka sobre "book" en una de las categorias que se hayan enseñado, se registrará la ID del cliente que esta guardada en la variable global "$_SESSION" para relacionar al cliente con la categoria elegida y registrar una reserva más tarde. Después, se hará un "INSERT", a la base de datos, con los datos recogidos del cliente y de la caregoria seleccionada.

Cuando el admin clicka en "book" la ID de cliente, que se usará más tarde para insertar datos, será una aleatoria de entre los clientes que están registrados en la base de datos. Esto fue pensado así para poder añadir reservas desde "Booking.php" con la perspectiva de cliente

SQL Docs

Este código está guardado en el documento de "039_hotel.sql".

El objetivo de este procedimiento es conseguir la lista de categorias que tengan, como mínimo, 1 habitacion disponible. Después, se guardarán en una tabla que, más tarde, será mostrada en la página "Booking".
